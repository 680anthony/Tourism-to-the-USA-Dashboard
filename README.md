# Tourism in the USA Analytics Dashboard

## Overview
Interactive Power BI dashboard that analyzes tourism and travel patterns into the United States. Explore international travel trends by country, region, visa type, entry method, state, month, and year.

## Purpose
The goal of this project is to convert raw tourism and border entry data into a structured analytical dashboard that provides insights into:
- International visitor trends
- Visa category distribution
- Entry methods (air, land, vehicle)
- Regional and country-level travel patterns
- Seasonal tourism behavior

## Tools & Technologies
- Power BI
- Power Query
- Alteryx
- UiPath (web scraping automation)
- Excel

## Data Sources
Data was collected from publicly available U.S. government sources:
- National Travel and Tourism Office (NTTO)
- U.S. Customs and Border Protection (CBP)

The dataset includes:
- International visitor counts
- Visa types (business, student, pleasure)
- Entry methods
- Country and regional breakdowns
- Monthly and yearly travel trends
- U.S. state-level entry data

## Data Processing
- UiPath was used to automate web data extraction
- Power Query and Alteryx were used for cleaning, transformation, and structuring
- Data models were built in Power BI to enable interactive filtering and relationships

## Dashboard Features
- Interactive filtering by year, country, region, visa type, and entry method
- Visa distribution analysis (business, student, pleasure)
- Entry method breakdown (air, vehicle, etc.)
- Geographic visualization of U.S. entry points by state
- KPI tracking for total visitor counts and travel composition

## Key Insights
- Pleasure travel represents the majority of international visits
- Air travel is the dominant entry method into the United States
- Tourism demand varies significantly by region and season
- Interactive filtering enables detailed country-level behavioral analysis
